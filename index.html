Porque jugar BALLMAZE

Este proyecto nace para poder desconectarse luego de un largo día de trabajo. Desafía a tus amigos a llegar al cofre lo más rápido posible. El cronómetro dirá quién es el más rápido.

Desarrollo BALLMAZE

Para el desarrollo de BALLMAZE se usó AppInventor. Haciendo uso del sensor de orientación tú puedes guiar la pelota hasta la llave y al cofre. Además el reloj te permite saber cuánto demoras en llegar a el cofre.

Desarrollo BALLMAZE web

La página web se desarrollá en base a una plantilla de html5up.net, sobre la cuál comenzamos a modificar el cádigo con ATOM para personalizarla y poder convertirla en la página que ahora alberga BALLMAZE
